MagicKeyboard

ELEC 327 Final Project

Oscar Zhang, Gerald Lu, Kerry Xu

Rice University, Department of Electrical and Computer Engineering

1. Introduction

The MagicKeyboard is a low-power embedded system designed to translate motion and touch into digital input signals. Built on the TI MSPM0 microcontroller platform, our system features real-time feedback, interrupt-driven control, and low-power design practices. We used capacitive touch sensors and gyroscopic input to interpret user intention, driving an output that mimics a keyboard.
