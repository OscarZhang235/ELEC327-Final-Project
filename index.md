## Project Description

This project aims to develop a low-power embedded system that interacts with the physical world in real time. Our system, designed using the TI MSPM0 microcontroller, performs closed-loop interaction based on sensory input and executes real-time control logic efficiently.

The final design demonstrates real-time responsiveness, modular state-based software design, and energy-efficient operation. We highlight hardware-software co-design principles, using timers, interrupts, PWM, and I/O peripherals of the MCU.

## Pedagogical Elements

The project utilizes:
- **TI MSPM0 MCU** for its low-power features and flexible peripherals
- **Timers and interrupts** for precise control
- **PWM control** for actuation
- **LFSR** for pseudo-random sequence generation
- **Custom game logic** for user interaction and feedback

The codebase is well-commented to serve as a reference for future embedded systems courses or hobbyists.
